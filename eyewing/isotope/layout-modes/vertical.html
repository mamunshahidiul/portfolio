Options

horizontalAlignment

Type: Number a decimal 0 to 1

Aligns items horizontally. 0 will align the origin edge. 1 will align the opposite edge. 0.5 will align center.

layoutMode: 'vertical',
vertical: {
  horizontalAlignment: 0.5,
}
